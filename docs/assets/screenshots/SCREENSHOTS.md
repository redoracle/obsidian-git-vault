# Screenshot Assets

This directory contains generated UI screenshots used by the README and documentation. Additional README images live under the repository-level `images/` directory.

## Regeneration

Run the Playwright screenshot harness from the repository root:

```bash
make docs-screenshots
```

Equivalent command:

```bash
cd screenshots
npm test
```

The harness regenerates:

- `docs/assets/screenshots/*.png`
- mirrored line-author PNGs under `docs/assets/line-author-*.png`
- selected README images under `images/`

Line-author GIF animations are manual recordings and are not generated by this harness.

## Standards

| Property   | Standard                                                          |
| ---------- | ----------------------------------------------------------------- |
| Resolution | 2x retina output via `deviceScaleFactor: 2`                       |
| Theme      | Obsidian default dark theme                                       |
| Format     | PNG for generated screenshots                                     |
| Naming     | Lowercase, hyphen-separated, descriptive                          |
| Data       | Placeholder content only; no personal tokens, paths, or user data |
