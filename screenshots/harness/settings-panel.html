Obsidian Git Vault v1.0.0
General
UI Mode
Simple mode shows a single Sync button. Advanced mode gives full Git control.
Sync Backend
Git (CLI) uses a local Git install. Gitless uses the GitHub API – no Git required.
GitHub Credentials
GitHub Token
Personal access token with repository read/write scope.
GitHub Owner
Your GitHub username or organisation name.
Repository
Name of the repository that stores your vault.
Branch
Branch to sync with. Defaults to main.
Conflict Resolution
Default Strategy
How to handle merge conflicts when both sides changed the same file.
Smart Sync Triggers
Sync on file change
Automatically sync shortly after you stop editing.
Sync on vault close
Push any pending changes when Obsidian closes.
Sync on network reconnect
Resume syncing automatically when you come back online.
Sync after idle (minutes)
Sync when you haven't made changes for this many minutes. Set to -1 to disable.
